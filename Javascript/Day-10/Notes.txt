2. Nominculture/ Rules

    a. Variable/function Naming Rules

        i. Allowed characters:
            Letters (a–z, A–Z), digits (0–9), Underscore _, All special characters (@, #, $)

        ii. Must NOT start with a number
            ❌ 1name
            ✅ name1

        iii. No spaces allowed
            ❌ user name
            ✅ userName

        iv. Case-sensitive
            age ≠ Age

        v. Cannot use reserved keywords
            ❌ let, function, return,... so on


    b. Different cases

        i. camelCase

            First word small, next words start with capital letter. No space between words

            Example:
                userName
                totalAmount

            Used for:
                Variables, Functions

        b. PascalCase

            Every word starts with capital letter. No space between words

            Example:
                UserName
                CalculateTotal

            Used for:
                Classes, Constructors, React Components

        c. snake_case

            Words separated by underscore "_" . No space between words

            Example:
                user_name
                total_amount

            Used for:
                Mostly backend, databases

        d. kebab-case

            Words separated by hyphen "-" . No space between words

            Example:
                user-name
                main-header

            Used for:
                File names, CSS classes

        e. Uppercase
        f. Lowercase


3. String methods

    1. .length
        Gives number of characters

        Example:
            str.length

    2. .toLowerCase()
        Converts string to lowercase

        Example:
            str.toLowerCase()

    3. .toUpperCase()
        Converts string to uppercase

    4. .trim()
        Removes extra spaces from start and end

    5. .charAt()
        Gets character at specific index

        Example:
            str.charAt(0)

    6. .indexOf()
        Finds position of character

    7. .includes()
        Checks if value exists

    8. .slice()
        Extracts part of string

        Example:
            str.slice(1, 4)

    9. .replace() , .replaceAll()
        Replaces part of string

    10. .split()
        Converts string into array

        Example:
            str.split(" ")


4. Array methods

    1. push()
        Adds element at the end of array

        Example:
            arr.push(10)

    2. pop()
        Removes last element

        Example:
            arr.pop()

    3. shift()
        Removes first element

        Example:
            arr.shift()

    4. unshift()
        Adds element at beginning

        Example:
            arr.unshift(5)

    5. length
        Gives total number of elements

        Example:
            arr.length

    6. indexOf()
        Finds index of element

        Example:
            arr.indexOf(10)

    7. includes()
        Checks if element exists

        Example:
            arr.includes(5)

    8. slice()
        Returns part of array (no change in original array)

        Example:
            arr.slice(1, 3)

    9. splice()
        Add/remove elements (changes original array)

        Example:
            arr.splice(1, 2)

    10. join()
        Converts array to string

        Example:
            arr.join("-")
