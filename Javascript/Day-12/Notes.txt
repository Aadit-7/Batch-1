2. Array methods

    1. .push()
        Adds element at the end of array

        Example:
            arr.push(10)

    2. pop()
        Removes last element

        Example:
            arr.pop()

    3. shift()
        Removes first element

        Example:
            arr.shift()

    4. unshift()
        Adds element at beginning

        Example:
            arr.unshift(5)

    5. length
        Gives total number of elements

        Example:
            arr.length

    6. indexOf()
        Finds index of element

        Example:
            arr.indexOf(10)

    7. includes()
        Checks if element exists

        Example:
            arr.includes(5)

    8. slice()
        Returns part of array (no change in original array)

        Example:
            arr.slice(1, 3)

    9. splice()
        Add/remove elements (changes original array)

        Example:
            arr.splice(1, 2)

    10. join()
        Converts array to string

        Example:
            arr.join("-")


Important concept that is "return"


3. JavaScript Functions Types

    1. Function Declaration (Named Function) (Pure / simple function)

        function greet() {
        console.log("Hello!");
        }


    2. Function Expression
        - Function stored in a variable

            const greet = function() {
            console.log("Hello!");
            };


    3. Arrow Function (ES6)  

        const greet = () => {
        console.log("Hello!");
        };


        Short form:

        const greet = () => console.log("Hello!");


    4. Callback Function

        - Passed as an argument to another function

            function processUser(name, callback) {
            callback(name);
            }

            processUser("Aadit", function(name) {
            console.log("Hello " + name);
            });


    5. Higher-Order Function

        - Takes function as argument OR returns a function


4. Advanced Array Methods


    1. reverse()
        Reverses array (modifies original)


    2. every()
        Returns true if ALL elements match

         Example:
            let arr = [1, 2, 3, 4, 5, 3, 6];

            console.log(
            arr.every((num) => {
                return num % 2 == 0;
            }),
            );


    3. some()
        Returns true if ANY element matches

        Example:
            let arr = [1, 2, 3, 4, 5, 3, 6];

            console.log(
            arr.some((num) => {
                return num % 2 == 0;
            }),
            );


    4. find()
        Returns first element that matches condition

        Example:
            let arr = [1, 2, 3, 4, 5, 3, 6];

            console.log(
            arr.find((num) => {
                return num % 2 == 0;
            }),
            );


    5. map()
        Creates a new array by transforming each element

        Example:
           let arr = [1, 2, 3, 4, 5, 6, 7];

            arr.map((num) => {
            console.log(num + 10);
            });


    6. filter()
        Returns elements that satisfy condition

        Example:
            let arr = [1, 2, 3, 4, 5, 6, 7];
            let arr2 = arr.filter((num) => {
            return num % 2 == 0;
            });

            console.log(arr);
            console.log(arr2);


    7. reduce()
        Reduces array to single value

        Example:
            arr.reduce((acc, curr) => acc + curr, 0)

    8. forEach()
        Loops through array (no return)

        Example:
            arr.forEach(num => console.log(num))


    9. sort()
        Sorts array (modifies original array)

        Example:
            arr.sort((a, b) => a - b)


    Method	Returns
    map	     new array
    filter	 new array
    reduce	 single value
    forEach	 nothing ❌

    iii. Original array vs new array
    Method	   Changes original?
    map	       ❌
    filter	   ❌
    reduce	   ❌
    sort	   ✅
    reverse	   ✅
