1. String methods

    1. .length
        Gives number of characters

        Example:
            str.length

    2. .toLowerCase()
        Converts string to lowercase

        Example:
            str.toLowerCase()

    3. .toUpperCase()
        Converts string to uppercase

    4. .trim()
        Removes extra spaces from start and end

    5. .charAt()
        Gets character at specific index

        Example:
            str.charAt(0)

    6. .indexOf()
        Finds position of character

    7. .includes()
        Checks if value exists

    8. .slice()
        Extracts part of string

        Example:
            str.slice(1, 4)

    9. .replace() , .replaceAll()
        Replaces part of string

    10. .split()
        Converts string into array

        Example:
            str.split(" ")

    11. String Concatination and template litreals


2. Array methods

    1. .push()
        Adds element at the end of array

        Example:
            arr.push(10)

    2. pop()
        Removes last element

        Example:
            arr.pop()

    3. shift()
        Removes first element

        Example:
            arr.shift()

    4. unshift()
        Adds element at beginning

        Example:
            arr.unshift(5)

    5. length
        Gives total number of elements

        Example:
            arr.length

    6. indexOf()
        Finds index of element

        Example:
            arr.indexOf(10)

    7. includes()
        Checks if element exists

        Example:
            arr.includes(5)

    8. slice()
        Returns part of array (no change in original array)

        Example:
            arr.slice(1, 3)

    9. splice()
        Add/remove elements (changes original array)

        Example:
            arr.splice(1, 2)

    10. join()
        Converts array to string

        Example:
            arr.join("-")
