JavaScript Basics (Simple Notes)

1. What is JavaScript?
   - JavaScript (JS) is a programming language used to make websites interactive.
   - It helps websites to respond for user actions like clicking buttons, filling forms, or loading new data.

        In simple words:
        HTML = structure (Skeloton)
        CSS = design (External Beauty)
        JavaScript = behavior (Brain)


2. History of JavaScript

* JavaScript was created in 1995 by Brendan Eich.
* It was developed in just 10 days.
* Initially, it was called "Mocha", then "LiveScript", and finally "JavaScript".
* Today, it is one of the most popular programming languages in the world.


3. Why JavaScript is Required?
   JavaScript is important because:

    * It makes websites interactive
    * It handles user actions (click, submit, input)
    * It allows communication with servers (APIs)
    * It is used in frontend (React) and backend (Node.js)

    Without JavaScript, websites would be static (no interaction).


4. How to Run JavaScript?
   You can run JavaScript in 2 main ways:

    a. Browser Console:

        * Open any website
        * Right click → Inspect → Console
        * Write code like:
        console.log("Hello")

    b. Using HTML file:

        * Write JS inside <script> tag or link external file
        * Open the HTML file in browser


5. How to Connect JavaScript with HTML File?

    a. Internal JavaScript:

        <script>
        console.log("Hello");
        </script>

    b. External JavaScript:

        * Create a file: script.js
        * Write JS code inside it
        * Connect it to HTML:

        <script src="script.js"></script>


6. What is the difference between var, let, and const?

    - var:

        * Old way to declare variables
        * Can be re-declared and updated
        * Function-scoped (less safe)

    - let:

        * Modern way
        * Can be updated but NOT re-declared in same scope
        * Block-scoped (safer than var)

    - const:

        * Used for fixed values
        * Cannot be updated or re-declared
        * Block-scoped

        Example:
        var a = 10;
        let b = 20;
        const c = 30;


7. What are data types in JavaScript?

    Data types define what kind of data a variable holds.

    Main types:

    i. String → text
        let name = "Aadit";

    ii. Number → numbers
        let age = 23;

    iii. Boolean → true/false
        let isStudent = true;

    iv. Null → empty value (intentionally)
        let x = null;

    v. Undefined → value not assigned
        let y;

    vi. Object → collection of data
        let person = { name: "Aadit", age: 23 };

    vii. Array → list of values
        let arr = [1, 2, 3];


8. What is the difference between null and undefined?

    - undefined:
        * Variable is declared but no value is assigned
    let a;
    console.log(a); // undefined

    - null:
        * Value is intentionally set as empty
        let b = null;

    Simple meaning:
    undefined = value not given
    null = value given but empty


9. How to see output?

    console.log   → Normal output
    console.table → Table format data
    console.warn  → Warning message
    console.error → Error message
    console.clear → Clears console
