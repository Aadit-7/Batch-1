1. What is the difference between var, let, and const?

    - var:

        - Old way to declare variables
        - Can be re-declared and updated
        - Function-scoped (less safe)

    - let:

        - Modern way
        - Can be updated but NOT re-declared in same scope
        - Block-scoped (safer than var)

    - const:

        - Used for fixed values
        - Cannot be updated or re-declared
        - Block-scoped

        Example:
        var a = 10;
        let b = 20;
        const c = 30;


2. What are data types in JavaScript?

    Data types define what kind of data a variable holds.

    Main types:

    i. String → text
        let name = "Aadit";

    ii. Number → numbers
        let age = 23;

    iii. Boolean → true/false
        let isStudent = true;

    iv. Null → empty value (intentionally)
        let x = null;

    v. Undefined → value not assigned
        let y;

    vi. Object → collection of data
        let person = { name: "Aadit", age: 23 };

    vii. Array → list of values
        let arr = [1, 2, 3];


3. What is the difference between null and undefined?

    - undefined:
        * Variable is declared but no value is assigned
    let a;
    console.log(a); // undefined

    - null:
        * Value is intentionally set as empty
        let b = null;

    Simple meaning:
    undefined = value not given
    null = value given but empty


4. How to see output?

    console.log   → Normal output
    console.table → Table format data
    console.warn  → Warning message
    console.error → Error message
    console.clear → Clears console


5. Operators in JavaScript

    Operators are used to perform operations on values.

    Types:

    i. Arithmetic Operators:

    Addition (+)
    Subtraction (-)
    Multiplication (*)
    Division (/)
    Modulus (%)


6. Conditional Statements (Decision Making)

    Used to run code based on conditions.

    i. if statement:

        let age = 18;
        if (age >= 18) {
        console.log("You can vote");
        }

    ii. if-else:
        let age = 16;
        if (age >= 18) {
        console.log("Adult");
        } else {
        console.log("Minor");
        }

    iii. else if:
        let marks = 75;
        if (marks >= 90) {
        console.log("A");
        } else if (marks >= 60) {
        console.log("B");
        } else {
        console.log("C");
        }


7. Functions in JavaScript

    Functions are reusable blocks of code.

    Example:
        function greet() {
        console.log("Hello Aadit");
        }
        greet();

    With parameters:
        function add(a, b) {
        return a + b;
        }

    console.log(add(2, 3)); // 5
