CSS PROPERTIES:-

1. flex wrap
    - flex-wrap: nowrap; (default) - all items will be on a single line
    - flex-wrap: wrap; - items will wrap onto multiple lines, from top to bottom
    - this property is givent to the parent element of the flex items, not to the child elements.

2. flex shrink
    - flex-shrink: 1; (default) - items will shrink to fit the container if necessary
    - flex-shrink: 0; - items will not shrink, they will overflow the container if necessary
    - this property is given to the child elements of the flex container, not to the parent

3. flex grow
    - flex-grow: 0; (default) - items will not grow to fill the container
    - flex-grow: 1; - items will grow to fill the container, sharing the available space equally
    - this property is given to the child elements of the flex container, not to the parent


4. --webkit-scrollbar
    - ::-webkit-scrollbar {
        width: 10px;
    }
    - ::-webkit-scrollbar-thumb {
        background-color: #888;
        border-radius: 5px;
    }


5. Positioning

    static
        - default position

    absolute
        - make that element position based on the nearest positioned ancestor (not static)
        - if there is no positioned ancestor, it will be based on the initial containing block (usually the viewport)

    relative
        - make child element position based on this element

    fixed
        - sticks to the given position even when the page is scrolled

    sticky;
        - we have to make this div after some scrollable content to see the effect of sticky.
        - behaves like relative until a certain scroll position is reached, then it becomes fixed

6. z-index
    - controls the stacking order of elements
    - higher z-index means the element will be in front of elements with lower z-index
